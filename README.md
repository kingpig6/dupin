# 獨品工坊開單系統

客製重機彩繪訂單管理 PWA，連接 Google 試算表。

## 快速啟動（5 步驟）

### 步驟 1：確認試算表工作表欄位

確認你的試算表有以下 4 個工作表，並在第 1 列填好欄位標題：

**訂單**
```
訂單編號 | 客戶 | 開單日期 | 交貨期限 | 狀態 | 收款狀態 | 備註
```
> ⚠️ 你現有的表缺「收款狀態」欄，請在 F 欄標題補上「收款狀態」，G 欄改成「備註」。

**客戶**
```
客戶名稱 | 聯絡人 | 電話 | 統一編號 | 地址 | 備註
```

**品項**
```
品項ID | 訂單編號 | 品名 | 規格 | 數量 | 單價 | 金額 | 車號 | 負責師傅
```

**設定**（第 A 欄=鍵，第 B 欄=值，逐列填入）
```
廠商名稱  | 獨品工坊
負責人    | 李安晟
統一編號  | 95323326
電話      | 0919726434
LINE      | kingpig6
匯款銀行  | 玉山銀行 808 台中分行
匯款戶名  | 獨品工坊 李安晟
匯款帳號  | 1366940043038
稅率      | 0
```

---

### 步驟 2：部署 Apps Script

1. 開啟你的 Google 試算表
2. 上方選單 → **擴充功能** → **Apps Script**
3. 把 `Code.gs` 的內容全部貼上（取代原本的 `function myFunction()`）
4. 點「存檔」（Ctrl+S）
5. 點「部署」→「**新增部署**」
   - 類型：**Web 應用程式**
   - 執行身分：**我（your@gmail.com）**
   - 誰可以存取：**所有人（Anyone）**
6. 點「部署」→ 複製**部署網址**（長得像 `https://script.google.com/macros/s/AKfycb.../exec`）

> 之後修改 Code.gs 需要重新部署（「管理部署」→ 鉛筆編輯 → 版本選「新版本」）

---

### 步驟 3：填入 API 網址

開啟 `app.js`，找到第 5 行：
```js
const API_URL = 'YOUR_APPS_SCRIPT_URL_HERE';
```
把 `YOUR_APPS_SCRIPT_URL_HERE` 換成你剛才複製的部署網址。

---

### 步驟 4：部署網頁（GitHub Pages）

1. 把這個 repo 推到你的 GitHub
2. GitHub → Settings → Pages → Source 選 `main` 分支 `/ (root)`
3. 儲存後幾分鐘網址就可以用，例如：`https://yourname.github.io/dupin/`

---

### 步驟 5：手機加到主畫面（PWA）

- **Android（Chrome）**：開啟網址 → 右上角三點選單 → 「新增至主畫面」
- **iOS（Safari）**：開啟網址 → 下方分享按鈕 → 「加入主畫面」

## 功能說明

| 頁面 | 功能 |
|---|---|
| 訂單列表 | 依狀態分區（進行中 / 完工交貨），顯示客戶、金額、收款狀態 |
| 訂單詳細 | 查看品項、即時更改狀態、新增品項、產生請款單 |
| 開單 | 建立新訂單，自動產生訂單編號 |
| 客戶管理 | 新增/編輯/刪除常用客戶 |
| 業績統計 | 各客戶累計 + 自訂日期範圍查詢 |
| 請款單 | 一鍵預覽並列印/匯出 PDF（含民國年、品項表、廠商資訊） |

## 員工權限設定

系統支援 **老闆（admin）／員工（staff）** 兩種角色，需先設定 Google 登入：

1. 在 Google Cloud Console 建立 OAuth 用戶端 ID，貼到 `app.js` 的 `GOOGLE_CLIENT_ID`
2. Apps Script →「指令碼屬性」新增 `GOOGLE_CLIENT_ID`（值需與上面相同）
3. 試算表建立「**員工**」工作表，欄位：
   ```
   email | 姓名 | 角色 | 抽成比例
   ```
   - `角色` 填 `admin`（老闆，完整權限）或 `staff`（一般員工）
   - `抽成比例` 選填，僅「抽成」類型的傭金計算會用到（例如 0.1 代表 10%）
4. 「**工作項目**」工作表建議加上「**建立者**」欄位（放在最後一欄即可），系統會自動填入開單者的姓名/email，供事後查核是誰開的單

登入後，**員工帳號**只能看到：
- 「進行中」裡「負責師傅」欄位空白（尚未指派）的項目，且**唯讀**（無法修改進度、收款狀態，也不能編輯或刪除）
- 「業績」分頁會換成「我的傭金」：自己「完工尚未結款」與「已結款」的傭金/抽成，已結款可用「本月」按鈕或自訂日期區間查詢
- 可以正常使用「開單」建立新工作項目（會記錄自己是開單人）

**老闆帳號**維持完整權限（所有訂單、業績統計、損益報告、刪除、開請款單等）。
