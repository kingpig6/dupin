# 素材放這裡 (assets)

把 5 張**去背（透明背景）灰階 PNG** 上傳到這個資料夾，檔名要完全一致：

| 檔名 | 用途 | 變色 |
|---|---|---|
| `a1.png` | 固定底圖（面罩/通風口/固定部分），最底層 | ❌ |
| `b1.png` | 部件 1 — 一般色（霧面）灰階 | ✅ |
| `c1.png` | 部件 1 — 金屬色 灰階 | ✅ |
| `b2.png` | 部件 2 — 一般色（霧面）灰階 | ✅ |
| `c2.png` | 部件 2 — 金屬色 灰階 | ✅ |

規格：全部**同尺寸**且與 `config.canvas`（1200×1200）一致、**背景透明**、灰階保留中間調與高光。
