# 安全帽彩繪模擬器

獨品工坊的網頁端互動式安全帽彩繪模擬器。顧客可即時換色、切換一般色／金屬色材質，並透過網址分享設計或直接下單（設計圖以 Email 寄回管理後台）。

獨立子專案，與開單系統互不干擾。純 HTML5 + CSS3 + Vanilla JS + Canvas 2D，無框架。

## 檔案

| 檔案 | 說明 |
|---|---|
| `index.html` | 主頁 |
| `styles.css` | 深色科技極簡樣式 |
| `config.js` | **唯一需要維護**：部位/素材路徑、調色盤、分店、API 網址 |
| `engine.js` | 雙軌灰階渲染引擎（缺素材會自動產生佔位圖）|
| `app.js` | UI 互動、URL 參數、下單表單 |
| `Code.gs` | Apps Script 寄信後端 |
| `assets/` | 灰階 PNG 素材 |

## 素材命名規範

所有 PNG 請用**相同尺寸**（預設 1200×900，需與 `config.canvas` 一致），且各部位在畫布上的位置已對齊。灰階圖要保留中間調與高光，純白/純黑區塊吃不到顏色。

| 檔名 | 用途 |
|---|---|
| `b1.png` | 固定底圖／固定圖騰（不變色），最底層 |
| `b2.png` `b3.png` | **一般色**（霧面）灰階圖 |
| `c1.png` `c2.png` | **金屬色** 灰階圖 |

部位與材質的對應在 `config.js` 的 `parts` 設定：
- 圖騰 A（`b2`）：一般色 `b2.png` / 金屬色 `c1.png`
- 圖騰 B（`b3`）：一般色 `b3.png` / 金屬色 `c2.png`

> ⚠️ c1/c2 與部位的配對為假設值，請依實際素材在 `config.js` 調整 `matteSrc` / `metalSrc`。

## 本機預覽

```bash
cd simulator
python3 -m http.server 8000
# 瀏覽器開 http://localhost:8000
```

未放素材時，引擎會以程式化灰階佔位圖渲染，方便先驗證流程。

## 下單寄信設定（Apps Script）

1. 建立新的 Apps Script 專案，貼上 `Code.gs`。
2. 專案設定 → 指令碼屬性，新增 `ADMIN_EMAIL = 你的收件信箱`。
3. 部署 → 新增部署 → 網頁應用程式（執行身分：我；存取：所有人）。
4. 複製部署網址，填入 `config.js` 的 `apiUrl`。

## 網址分享機制

- 編碼：`?b2=C0C0C0_M&b3=1A1A1A_N`（`_M`=金屬色 / `_N`=一般色）。
- 改色時即時寫入網址；「複製分享網址」可複製目前設計。
- **無參數的基礎網址 = 預設外觀**。發給新客戶請用基礎網址，避免載到上一位的設計。
